## 주요 기능

- 미세먼지, 온도, 대기질지수 등 날씨에 대한 정보 확인
- 7일간 옷차림 및 기온이 확인이 가능하다.
- 카카오톡 로그인 및 공유 기능
- 위치를 네이버 지도API 설정 가능
- 알람 기능
- 다크 모드 지원
- 언어 설정 변경 가능 (영어)
